# html_css_js
# Basic HTML, CSS & JavaScript Website

## Description

This project is a simple responsive website developed using **HTML, CSS, and JavaScript**. It demonstrates the fundamental concepts of front-end web development by combining various HTML elements with CSS styling and JavaScript interactivity.

The webpage includes a navigation bar, headings, text formatting, hyperlinks, images, ordered and unordered lists, tables, a registration form with password validation, audio and video embedding, and a footer. CSS is used to provide a clean, responsive, and visually appealing layout, while JavaScript enhances user interaction through welcome messages, form validation, character counting, date and time display, and an optional dark mode feature.

### Features

* Responsive webpage layout
* Styled navigation bar and footer
* Text formatting examples
* Hyperlinks and images
* Ordered and unordered lists
* Student details table
* Registration form with password validation
* Audio and video embedding
* Welcome alert on page load
* Current date and time display
* Form submission validation
* Live character counter for the message field
* Optional Dark Mode toggle

### Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

### How to Run

1. Download or clone the project.
2. Place all required media files (images, audio, and video) in the project folder.
3. Open `index.html` in any modern web browser.
4. Explore the webpage and interact with its features.
